## 使用建议

后续进入 `single-satellite L1g` 后，可以优先做两类对照：

1. 视觉对照  
   对比 quicklook 结构、云系位置、日照边界、窗口区亮温形态是否一致。

2. 几何对照  
   对比海岸线、圆盘边界、区域覆盖、极区拉伸和边缘畸变是否发生异常变化。
